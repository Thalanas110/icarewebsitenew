# icarewebsite

This repository contains a Vite + React + TypeScript project scaffold.

## Getting started

Prerequisites: Node.js and npm (or pnpm/yarn).

Quick start:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## Technologies

- Vite
- React
- TypeScript
- Tailwind CSS

## Deployment

Deploy like any static frontend app (Netlify, Vercel, Surge, or static hosting).

If you want, I can add a short `README` section explaining a preferred deploy target.
